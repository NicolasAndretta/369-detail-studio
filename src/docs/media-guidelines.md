# 369 Detail — Media Guidelines

## Media Philosophy

Media is one of the most important parts of the 369 Detail experience.

Images and videos must feel:

- cinematic
- premium
- emotional
- immersive
- professional

The media should communicate quality instantly.

The website should feel visually curated.

Never generic.

---

# Visual Identity Standards

The visual language should feel inspired by:

- luxury automotive commercials
- cinematic garage photography
- premium detailing studios
- industrial lighting
- modern car culture

The atmosphere should feel:

- dark
- elegant
- masculine
- refined
- immersive

---

# Photography Style

Photography should prioritize:

- dramatic lighting
- contrast
- reflections
- textures
- paint detail
- cinematic framing

Important visual focus areas:

- glossy paint reflections
- ceramic coating shine
- wheel details
- interior precision
- water reflections
- lighting streaks

---

# Preferred Image Style

Preferred image characteristics:

- high contrast
- deep blacks
- soft green accents
- controlled highlights
- cinematic shadows
- shallow depth of field

Avoid:

- oversaturated colors
- excessive HDR
- cheap filters
- noisy compression
- low-quality mobile photos

---

# Before / After Content

Transformation content is critical.

Before/after imagery should communicate:

- precision
- improvement
- craftsmanship
- attention to detail

Best practices:

- same camera angle
- same lighting conditions
- clear transformation visibility

Avoid misleading edits.

Authenticity builds trust.

---

# Hero Media Standards

Hero visuals are extremely important.

Hero images/videos should feel:

- cinematic
- luxurious
- emotionally impactful
- clean
- atmospheric

Avoid crowded visuals.

The hero should instantly create desire and curiosity.

---

# Video Guidelines

Videos should feel:

- smooth
- atmospheric
- premium
- cinematic

Preferred video styles:

- slow motion
- detailing closeups
- reflection shots
- water movement
- polishing sequences
- cinematic garage movement

Avoid:

- shaky handheld footage
- low lighting quality
- noisy audio
- chaotic editing

---

# Media Compression Standards

All media must remain optimized.

Preferred formats:

Images:

```txt
.webp
.avif
```

Videos:

```txt
.mp4
.webm
```

Avoid oversized files.

Performance remains critical.

---

# File Naming Standards

All assets should use descriptive naming.

Examples:

```txt
ceramic-coating-closeup.webp
bmw-detailing-before-after.webp
369-detail-hero-video.mp4
```

Avoid:

```txt
IMG_4829.jpg
final-edit-new.png
video1.mp4
```

---

# Folder Organization

Media should remain organized by purpose.

Structure:

```txt
/images
  /hero
  /gallery
  /services
  /before-after
  /branding

/videos
```

Avoid random asset placement.

---

# Branding Assets

Branding materials include:

- logo
- icons
- overlays
- textures
- watermark elements

The logo should remain:

- sharp
- readable
- premium
- scalable

Avoid stretched or low-quality branding assets.

---

# Image Usage Rules

Images should support storytelling.

Every image must have a purpose.

Avoid:

- decorative clutter
- repetitive visuals
- generic stock photos
- random placeholders

The gallery should feel curated.

---

# Mobile Media Optimization

Media must remain optimized for mobile.

Priorities:

- responsive scaling
- compressed loading
- fast rendering
- minimal layout shift

Avoid oversized hero media on mobile.

---

# Accessibility Standards

Images should support accessibility.

Use:

```html
alt=""
```

for decorative assets.

Use descriptive alt text for important visuals.

Videos should avoid autoplay audio.

---

# Visual Consistency

All media should feel part of the same visual universe.

Avoid mixing:

- inconsistent editing styles
- random color grading
- different visual moods
- mismatched quality levels

The website should feel cohesive.

---

# Future Media Expansion

Future media systems may include:

- cinematic reels
- interactive galleries
- transformation sliders
- service videos
- branded photography sessions

Architecture should support future expansion.

---

# Final Philosophy

The media experience should feel:

- cinematic
- emotional
- luxurious
- immersive
- high-end

The user should instantly feel:

"This is premium automotive care."
