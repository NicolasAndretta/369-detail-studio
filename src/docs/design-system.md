# 369 Detail — Design System

## Design Philosophy

The visual identity of 369 Detail must feel cinematic, premium, industrial, and emotionally impactful.

This is not a generic automotive website.

The interface should communicate:

- luxury
- precision
- darkness
- confidence
- transformation
- obsessive detailing culture

The design language should feel inspired by:

- premium automotive commercials
- luxury garages
- modern industrial interiors
- cinematic vehicle photography
- underground premium culture

The website should feel immersive rather than corporate.

---

# Core Visual Identity

## Primary Mood

The atmosphere must feel:

- dark
- masculine
- premium
- refined
- cinematic
- modern
- aggressive but elegant

Avoid:

- playful aesthetics
- startup visuals
- SaaS-style layouts
- cartoon colors
- exaggerated cyberpunk effects
- noisy interfaces

---

# Color System

## Primary Colors

### Deep Black

Used as the main background color.

```css
#050505
```

Used for:

- page backgrounds
- hero overlays
- cinematic sections
- footer areas

---

## Graphite

Secondary dark surface.

```css
#111111
```

Used for:

- cards
- navbar glass backgrounds
- elevated sections
- modal surfaces

---

## Soft Carbon

Used for subtle contrast layers.

```css
#1A1A1A
```

Used for:

- hover states
- secondary cards
- layered gradients

---

## Chicago Green

Primary accent color inspired by the local identity of Nueva Chicago without becoming sports-oriented.

```css
#0D3B2A
```

This color must be used carefully and strategically.

Use for:

- glow accents
- CTA highlights
- hover states
- section separators
- premium gradients

Avoid excessive saturation.

---

## Metallic Silver

Used for:

- subtle typography accents
- separators
- premium reflections
- icon accents

```css
#B8B8B8
```

---

# Border Philosophy

Borders should remain subtle and atmospheric.

Preferred border style:

```css
border-white/10
```

Avoid:

- thick borders
- harsh outlines
- fully opaque borders

---

# Shadows & Glow

The interface should use cinematic shadows.

Preferred:

- soft shadows
- layered shadows
- ambient green glows
- subtle reflections

Avoid:

- hard drop shadows
- cartoon glow
- neon cyberpunk effects

---

# Typography Direction

Typography must feel premium and modern.

Recommended fonts:

- Inter
- Outfit
- Geist
- Satoshi

Headlines should feel:

- bold
- cinematic
- spacious
- luxurious

Body text should remain:

- clean
- readable
- soft contrast

Preferred utilities:

```css
tracking-tight
leading-relaxed
text-white/80
```

---

# Spacing Philosophy

The website must breathe.

Use generous spacing between:

- sections
- cards
- text blocks
- CTAs

Avoid cramped layouts.

Preferred section spacing:

```css
py-24
lg:py-32
```

---

# Motion Philosophy

Animations should feel:

- smooth
- premium
- cinematic
- intentional

Never:

- overly bouncy
- chaotic
- exaggerated

Preferred animation style:

- fade
- slow reveal
- blur reveal
- stagger entrance
- smooth parallax

Preferred timing:

```css
duration-500
ease-out
cubic-bezier(0.16, 1, 0.3, 1)
```

---

# Visual Composition

The website should feel like:

- a premium automotive commercial
- a luxury garage
- a cinematic transformation studio

Each section should feel visually distinct while remaining cohesive.

The user should feel:

- trust
- exclusivity
- aspiration
- professionalism
- obsession for detail

---

# UI Direction

## Navbar

The navigation bar should:

- remain transparent on top
- become glassmorphic on scroll
- use subtle blur
- feel lightweight and premium

Avoid oversized navigation layouts.

Preferred style:

- minimal
- cinematic
- floating effect
- soft borders
- premium spacing

---

## Buttons

Buttons must feel premium and tactile.

Preferred behavior:

- smooth hover transitions
- subtle glow
- slight scale interactions
- elegant shadows

Avoid:

- generic gradients
- oversized rounded corners
- default Tailwind button appearance

CTA buttons should feel:

- luxurious
- confident
- modern

---

## Cards & Sections

Cards should use:

- soft contrast
- atmospheric borders
- layered depth
- premium spacing

Avoid flat interfaces.

Preferred utilities:

```css
backdrop-blur-md
bg-white/5
border border-white/10
```

---

# Responsive Philosophy

The experience must feel premium on:

- mobile
- tablet
- desktop
- ultrawide screens

Mobile experience is not secondary.

Animations and layouts must scale correctly without breaking immersion.

Avoid:

- cramped mobile layouts
- oversized desktop text
- horizontal scrolling

---

# Conversion Psychology

The website is not only informational.

It must psychologically communicate:

- professionalism
- trust
- exclusivity
- transformation
- premium service quality
