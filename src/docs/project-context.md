# 369 Detail — Project Context

## Brand Overview

369 Detail is a premium automotive detailing studio based in Lugano / Mataderos, Buenos Aires, Argentina.

The brand combines:

- premium detailing culture
- cinematic automotive presentation
- obsessive attention to detail
- neighborhood identity
- masculine luxury aesthetics

This is not a cheap car wash business.

The objective is to position 369 Detail as a high-value automotive care studio focused on:

- paint correction
- ceramic coatings
- premium detailing
- visual transformation
- personalized customer attention

The communication style must feel:

- premium
- elegant
- cinematic
- aggressive but refined
- trustworthy
- modern
- confident

The website should emotionally communicate:
"Your vehicle is treated like a high-end machine."

---

## Business Philosophy

369 Detail focuses on quality over volume.

The experience should feel exclusive even for regular vehicles.

The brand must communicate:

- professionalism
- precision
- obsession with details
- transformation
- visual excellence

Customers should feel:

- confidence
- pride
- excitement
- aspiration

The project should never feel:

- generic
- cheap
- overcrowded
- amateur
- overly colorful
- football-themed

Neighborhood identity exists subtly through color language and attitude, not explicit sports branding.
