# 369 Detail — SEO Strategy

## SEO Philosophy

The SEO strategy for 369 Detail must prioritize:

- local visibility
- premium brand positioning
- fast loading performance
- mobile-first indexing
- conversion-oriented traffic

The website should rank for users searching for premium automotive detailing services in:

- Lugano
- Mataderos
- CABA
- surrounding zones

The site must feel premium both to users and search engines.

Descriptions must:

- sound premium

# Performance SEO

## Core Web Vitals Priority

The site must optimize:

- Largest Contentful Paint (LCP)
- Interaction to Next Paint (INP)
- Cumulative Layout Shift (CLS)

---

## Performance Rules

### Images

All images must:

- use WebP or AVIF
- include width and height
- support lazy loading
- remain optimized

Avoid:

- oversized JPEG files
- uncompressed assets
- layout shifts

---

### Fonts

Use optimized font loading.

Prefer:

- variable fonts
- `next/font`
- minimal font families

Avoid:

- excessive font weights
- unnecessary font requests

---

The website must feel exceptional on mobile devices.

Priorities:

- large tap targets

- fast interaction
- smooth scrolling
- optimized spacing

The mobile version is the primary experience.

# URL Structure

## Clean URLs

URLs must remain:

- short
- descriptive
- lowercase
- readable

Examples:

```txt
/services
/gallery
/contact
```

Avoid:

```txt
/page?id=14
```

---

# Image SEO

## Naming Conventions

Images should use descriptive names.

Correct:

```txt
ceramic-coating-bmw.webp
before-after-detailing-audi.webp
```

Avoid:

```txt
IMG_9382.jpg
```

---

## Alt Text Standards

All meaningful images must include descriptive alt text.

Example:

```html
alt="Black BMW after premium ceramic coating treatment"
```

Decorative images may use:

```html
alt=""
```
